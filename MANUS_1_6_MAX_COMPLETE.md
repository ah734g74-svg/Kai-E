# Manus 1.6 Max - DarkForge-X الإصدار الخارق

## نظرة عامة
تم ترقية تطبيق DarkForge-X إلى مستوى **Manus 1.6 Max** بالكامل، مع إضافة أنظمة متقدمة وخوارزميات فائقة السرعة والدقة.

---

## 1. نظام الموصلات (Connectors)
**الملفات:**
- `connectors/ConnectorModels.kt` - نماذج البيانات للموصلات
- `connectors/ConnectorManager.kt` - مدير الموصلات

**الميزات:**
- دعم GitHub, Google, Shopify, Outlook, Instagram, Meta, Browser
- إدارة الصلاحيات والاتصالات برمجياً
- تفعيل/تعطيل الموصلات بسهولة

---

## 2. محرك التطور الذاتي (Self-Evolution Engine)
**الملفات:**
- `data/SelfEvolutionEngine.kt` - محرك التطور الذاتي

**الميزات:**
- تدقيق الكود قبل التنفيذ (Static Analysis)
- محاكاة التنفيذ في بيئة معزولة (Dry Run)
- التحسين التلقائي للأداء (Self-Optimization)
- مراقبة الأداء الفعلي (Runtime Monitoring)
- تسجيل الأخطاء والتعلم منها

---

## 3. محرك تنفيذ الأوامر Omega
**الملفات:**
- `data/OmegaExecutionEngine.kt` - محرك Omega

**الميزات:**
- تنفيذ الأوامر بأقصى سرعة ودقة
- توازي فائق (Hyper-Parallelism)
- إدارة ذكية للمكتبات (apk, pip, npm)
- تدقيق فوري قبل التنفيذ

---

## 4. نظام Linux الجديد (Omega Linux)
**الملفات:**
- `sandbox/OmegaLinuxManager.kt` - مدير Omega Linux الجديد

**الميزات:**
- استبدال النظام القديم بنظام أسرع وأقوى
- دعم كافة المكتبات الحديثة والقديمة
- قائمة مكتبات شاملة:
  - أساسيات: bash, curl, wget, git, python3, nodejs
  - الشبكات: openssh, nmap, tcpdump, socat
  - التطوير: build-base, libffi-dev, openssl-dev
  - المتقدمة: postgresql-client, redis, graphviz
  - معالجة الملفات: imagemagick, ffmpeg, ghostscript
  - أدوات النصوص: vim, nano, grep, sed, awk, perl
  - الضغط: zip, tar, gzip, bzip2, xz
  - أدوات النظام: htop, gdb, strace, ltrace

---

**الملفات:**
- `cloud/CloudComputerManager.kt` - مدير الكمبيوتر السحابي

**الميزات:**

- مراقبة استخدام الموارد (CPU, Memory, Storage)

## 6. نظام الـ API المتقدم
**الملفات:**
- `api/AdvancedApiManager.kt` - مدير الـ API المتقدم

**الميزات:**
- إدارة مفاتيح الـ API (API Keys)
- دعم مفتاح الربط التاريخي (Legacy API Key): `2004-06-29-manus-legacy-api-key`
- إدارة نقاط النهاية (Endpoints)
- دعم أنواع المصادقة المختلفة (Bearer, API Key, OAuth2)
- التحقق من صحة المفاتيح

**مفتاح الربط التاريخي:**
- **ID:** `legacy-2004-06-29`
- **Value:** `2004-06-29-manus-legacy-api-key`
- **التاريخ:** 2004-06-29 00:00:00 UTC
- **الحالة:** نشط (Active)
- **الوصف:** Legacy API Key from Manus 1.0 era
